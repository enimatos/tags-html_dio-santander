🧱 O Que é HTML? HTML significa HyperText Markup Language, ou em português, Linguagem de Marcação de Hipertexto. Ela é a linguagem usada para criar páginas da web. Com o HTML, nós escrevemos o conteúdo que será exibido em um navegador, como textos, imagens, vídeos, links, tabelas e muito mais. Ele não é uma linguagem de programação (como JavaScript ou Python), porque não executa ações ou cálculos. O HTML serve para organizar e estruturar as informações de uma página. 📄 Por exemplo: Quando você acessa um site e vê um título, um parágrafo, um botão ou uma imagem, tudo isso foi colocado ali com HTML.
